Reactive v1.0.3 Features

Complete showcase of the latest template parser features, modifiers, and structural directives.

1. Constant Expressions

Expressions that don't depend on state are evaluated once at mount.

{{ 100 + 200 }} = {{ 100 + 200 }}
2. JavaScript Operations

Run any safe JS directly in your templates.

Math.PI.toFixed(4) {{ Math.PI.toFixed(4) }}
Array Mapping {{ [1,2,3].map(x => x * 10).join(', ') }}
3. Event Modifiers & Key Handling
@click.once
@keydown.enter
Last key: {{ app.lastKey }}
4. Debouncing

Prevents rapid updates (useful for search or expensive logic).

@input.debounce.500
Search: {{ app.search }}
:model.debounce.500
Model: {{ app.debouncedModel }}
5. Input Modifiers
:model.number (automatic casting)
Type of age: {{ typeof app.age }} ({{ app.age }})
:model.trim (automatic trim)
Username: "{{ app.username }}"
6. Structural Directives (:if & :for)

Declarative conditional and list rendering.

:if conditional
Admin Panel Visible
Standard View
:for loop
  • {{ user.name }} ({{ user.role }})
